## Privacy Statement

The CS Calculator 2.0 does not store any user data. This includes SPD data, calculation data, personally identifiable data, and etc. The CS Calculator 2.0 does not use cookies to store data on the client side, nor does it connect to any services which would allow it to store data on our side, or with any third parties. All calculation data is processed on the user's local machine.
